# Dashboard

PumpSteer supports:

- mini-graph-card  
- apexcharts-card  

---

## What you can visualize

- Indoor temperature  
- Target temperature  
- Virtual outdoor temperature  
- Price behavior  

---

## Setup

1. Install cards via HACS  
2. Paste YAML into dashboard  
3. Adjust entity IDs  

---

## Tips

- No graph → check entity IDs  
- No data → check sensors
