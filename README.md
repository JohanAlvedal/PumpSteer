# 🔥 PumpSteer – Smart Heat Pump Optimization

➡️ Swedish version: [README (Svenska)](README_sv.md)

## 🚀 Latest release: 2.1.0  
👉 See [Changelog](docs/CHANGELOG.md)

PumpSteer is a Home Assistant custom integration that optimizes your heat pump by dynamically adjusting the **virtual outdoor temperature**.

It reduces energy cost when electricity is expensive — while protecting indoor comfort.

## 🔧 How it works

PumpSteer controls your heat pump by manipulating the **virtual outdoor temperature**.

Instead of controlling setpoints directly, it influences the system via the outdoor sensor input.

👉 See full explanation:  
[Configuration & System Overview](docs/CONFIGURATION.md)
